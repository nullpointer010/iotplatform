# Plataforma IoT - CropDataSpace

Este repositorio contiene la plataforma IoT desarrollada para el proyecto CropDataSpace. La plataforma está diseñada para gestionar y procesar datos de sensores agrícolas. Entre los servicios incluidos en la plataforma se encuentran:
- Un backend modular y escalable desarrollado con tecnologías modernas como CrateDB, Orion Context Broker, MongoDB y una API RESTful desarrollada con FastAPI. La plataforma permite la ingesta, almacenamiento, consulta y análisis de datos en tiempo real, facilitando la toma de decisiones informadas en el ámbito agrícola.
- Servicios de visualización mediante Apache Superset, que permiten a los usuarios crear dashboards interactivos y personalizados para monitorizar los datos de los sensores y obtener insights valiosos.
- Integración con herramientas de análisis de datos y machine learning como H2O.ai para potenciar el valor de los datos recopilados, facilitando la predicción de tendencias y la optimización de recursos agrícolas.
- Servicios de programación nocode como Node-RED, Apache NiFi y Apache Airflow para facilitar la creación de flujos de trabajo y automatizaciones sin necesidad de escribir código.
- Servicios de almacenamiento en la nube mediante MinIO, proporcionando una solución escalable y eficiente para gestionar grandes volúmenes de datos generados por los sensores agrícolas.
- Servicios de mensajería y comunicación en tiempo real mediante MQTT con Eclipse Mosquitto, permitiendo la transmisión eficiente de datos entre dispositivos y la plataforma IoT.
- Servicios de monitorización y logging con Prometheus y Grafana, que ofrecen capacidades avanzadas para supervisar el rendimiento de la plataforma y analizar logs en tiempo real.
- Servicios de autenticación y autorización con Keycloak, garantizando la seguridad y el control de acceso a los recursos de la plataforma IoT.
